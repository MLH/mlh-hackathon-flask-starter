Fetching your starred repos

You can request data from the GitHub API using the services/github.py module. This makes it easier to make authenticated requests.

To fetch data from the API, you can use this module to make a GET request.

    
      github = GitHub(access_token=session['access_token'])
      results = github.get('/user/starred')
